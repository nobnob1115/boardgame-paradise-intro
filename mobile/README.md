# ボードゲームパラダイス 会員システム 紹介資料（スマホ縦画面版）

iPhone縦画面（390×844）向けにレイアウトし直したスマホ専用の紹介資料です。

## GitHub Pagesでの公開手順

1. GitHubで新しいリポジトリを作成（Public・例：`boardgame-paradise-mobile`）
2. この中のファイルを全てアップロード（ドラッグ&ドロップでOK）
3. リポジトリの **Settings → Pages** を開く
4. 「Source」で **Deploy from a branch** を選択
5. Branch を `main` / root (`/`) に設定して **Save**
6. 1〜2分待つと以下のURLで公開されます：
   `https://<あなたのユーザー名>.github.io/boardgame-paradise-mobile/`

## 操作方法
- **スマホ**: 上下スワイプでスライド送り
- **PC**: ↑↓キー、スペース、マウスホイール
- スクロール位置は自動的に保存されます（localStorage）
